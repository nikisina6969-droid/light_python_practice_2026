# Индексатор папок (полный вариант)

Консольная утилита на Python для индексации папок. Учебная практика, полный
вариант: индекс файлов, хэши и результаты проверок хранятся в SQLite (`sqlite3`).
Используется только стандартная библиотека Python.

## Возможности
- рекурсивный обход папки и сбор метаданных (путь, размер, дата, тип);
- индекс файлов в SQLite (исчезнувшие файлы помечаются как `missing`);
- фильтр вывода по расширению и по имени;
- подсчёт SHA-256 и поиск дубликатов (с переиспользованием хэшей между запусками);
- сверка папки с резервной копией и история проверок.

## Запуск
```bash
python src/main.py <путь_к_папке> [опции]
```
При первом запуске создаётся база `data/app.db`.

### Примеры
```bash
# сканирование и вывод индекса
python src/main.py C:/Users/me/Documents

# фильтр по расширению / по имени
python src/main.py C:/Users/me/Documents --filter-ext .pdf
python src/main.py C:/Users/me/Documents --filter-name отчет

# поиск дубликатов
python src/main.py C:/Users/me/Documents --duplicates

# сверка с резервной копией + история проверок
python src/main.py C:/Users/me/Documents --backup D:/backup/Documents --history
```

## Структура
```
src/
  main.py         точка входа, разбор аргументов
  db.py           подключение и схема SQLite
  scanner.py      обход папки и сохранение индекса
  duplicates.py   хэши и поиск дубликатов
  backup.py       сверка с резервной копией
  report.py       вывод отчётов в консоль
  student_meta.md метаинформация
data/app.db       база (создаётся автоматически, в .gitignore)
journal.md        журнал работ по этапам
```

## Схема базы
- `files` — индекс файлов (путь, размер, mtime, тип, хэш, статус);
- `scans` — журнал запусков сканирования;
- `backup_checks` — сводка по каждой проверке резервной копии;
- `backup_diffs` — детали проверки (отсутствующие, изменённые, лишние файлы).
