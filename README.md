# Индексатор папок (полный вариант)

Консольная утилита на Python для индексации папок. Учебная практика, полный
вариант: используется SQLite (`sqlite3`) для хранения индекса файлов.

## Запуск

```bash
python src/main.py <путь_к_папке>
```

При первом запуске создаётся база `data/app.db`.
